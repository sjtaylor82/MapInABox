1. Overview

Map in a Box is an accessible world map and local exploration program designed for blind and low-vision users. It runs on Windows and macOS and works fully with screen readers including NVDA, JAWS, and VoiceOver.

The program combines global geography with street-level navigation. You can explore countries, cities, oceans, and landmarks from the world map, then drop into any street network to walk intersections, look up businesses, plan routes, and check departure boards — all without a visual display.

The program has six main modes:

ModeWhat it does
Map modeExplore the world map by country, city, coordinates, and geographic features. The default mode on startup.
Street modeExplore streets, intersections, house numbers, and nearby points of interest at the local level.
Walking modeMove along a real street network with controlled turn choices at intersections, with optional POI announcements.
Free modeExplore nearby streets and points of interest relative to a heading, without being locked to a road.
ToolsRoute planning, transit, flights, hotels, food, toll calculation, and journey planning.
Challenge gameA geography game that asks you to locate a named country on the world map by sound.

2. Screen reader notes

Map in a Box delivers most information through accessible speech output directly — spoken announcements do not require keyboard focus to be on any particular control. When the app speaks a location, distance, or status, it is intentional and does not require you to navigate to a list item to hear it.

Many screens present results in a list. Standard list behaviour applies:

KeyAction
Up / DownMove through items
EnterOpen, jump to, or select the current item
EscapeClose the list or dialog
F1Keyboard help for the current mode

If you are unsure where focus is or what mode you are in, press F2 in map mode to repeat the current location, or press F1 for context-sensitive keyboard help.

3. Starting out

When Map in a Box opens, it places you at your saved home location. If no home location has been set, the program prompts you to set one before continuing. Home can be changed at any time from the Settings dialog (Ctrl+,) using the Set Home Location button.

The main window contains a visual map canvas and a status area. Screen reader users interact primarily through spoken output and the keyboard — the map canvas is not required. All navigation, lookup, and mode-switching functions are keyboard driven.

5. Jumping to a place

Press J in map mode to open the jump dialog. Type a search term and press Enter. If multiple results are found, a list is presented — use the arrow keys to choose and press Enter to jump.

Accepted search terms include:

  • City names — e.g. Brisbane, London, Vancouver
  • Countries — e.g. Japan, New Zealand, South Africa
  • Coordinates — decimal degrees, e.g. -27.47, 153.03
  • Geographic features — major deserts, mountain ranges, reefs, gulfs, capes, and other named natural features

If the search term is not found, the program says "not found" and returns focus to the input field so you can try a different term. Jump is disabled while the geography challenge game is active.

7. Spatial tones

Spatial tones give you an audio sense of your position on the map without needing speech. Pitch represents latitude (higher pitch toward the north) and stereo panning represents longitude (left for west, right for east).

Three modes are available in Settings:

ModeDescription
WorldTones are calibrated to the entire world map. Best for global exploration.
CountryTones are scaled to the current country's bounding box. Gives finer resolution within a single country.
RegionTones are scaled to the current state, province, or region where available. Most useful in large countries for local navigation.

8. Marks and destination

Map mode provides three named mark slots and one destination slot. Marks and destination are independent positions you store for comparison or reference.

KeyAction
MOpen the mark menu. Press 1, 2, or 3 immediately to store the current position in that slot.
Shift+MOpen the remove mark menu. Press 1, 2, or 3 to clear that slot.
DStore the current position as the destination.
Alt+1Report distance and bearing from mark 1 to the destination.
Alt+2Report distance and bearing from mark 2 to the destination.
Alt+3Report distance and bearing from mark 3 to the destination.

A typical use is to mark three candidate pickup points, set a common destination (such as an airport), then compare direction and distance for each using Alt+1 through Alt+3.

9. Favourites

Favourites let you save and quickly return to frequently used places, points of interest, or transport stops.

KeyAction
Ctrl+FOpen the favourites list
Ctrl+Shift+FAdd the current map position, selected POI, or current location to favourites

Favourites are stored locally and persist across sessions.

11. Walking mode

Walking mode simulates moving along a real street network. You travel forward and backward along the current street and choose which way to turn at intersections. Nearby points of interest can be announced automatically as you walk.

POI announcements while walking are configured in Settings. You can choose the category of POIs to announce, the detection radius (50, 80, or 120 metres), and whether the category label is included in each announcement.

13. Points of interest

Points of interest (POIs) include businesses, transport stops, parks, medical services, accommodation, and landmarks. POI data is sourced from OpenStreetMap by default. A HERE API key enables richer live detail such as opening hours, phone numbers, and ratings.

When opening the POI list, you are prompted to choose a category:

  • All nearby
  • Food and drink
  • Shopping
  • Public transport
  • Trains and stations
  • Health and medical
  • Community and services
  • Arts, venues and landmarks
  • Parks and outdoors
  • Accommodation

Selecting a transit stop and pressing Ctrl+Enter opens departure information for that stop where timetable data is available.

14. Satellite and Street View

Map in a Box can open satellite imagery and street-level photographs for the current position. Both features require a Google API key. When a Gemini API key is also configured, the program requests an AI-generated verbal description of the image, which is read aloud.

Satellite view

Satellite view displays an overhead aerial image of the current position in an in-app dialog. The Gemini description appears in a readable text field below the image and is announced through the screen reader. The description covers terrain type, land use, built environment, water bodies, and vegetation. Without a Gemini key, the image is still shown but the text field reads: "Satellite imagery loaded. A Gemini API key is required to fetch a visual description."

Street View

Street View fetches up to two ground-level photographs of the current location — one in each direction along the street — and displays them in an in-app dialog. Below the images, the Gemini description is shown in a readable text field and announced through the screen reader. With a Gemini key, the description covers building types, street layout, signage, access features, and surroundings. Without a Gemini key, the images are still displayed but the text field reads: "Street View imagery loaded. A Gemini API key is required to fetch a visual description."

If no Google API key is configured, the program opens Mapillary in your browser as a fallback — a web-based open street-level viewer. No in-app dialog or description is provided in this case. If Google Street View has no coverage at the current location, the program automatically falls back to satellite view instead.

Note: Both satellite and Street View open inside the app as dialogs, not in a browser. The Gemini description is announced by the screen reader as well as shown in the dialog text field. The Mapillary browser fallback (no Google key) is the only case where a browser opens.

15. Find food

The Find Food feature searches for food and drink venues along a route between a saved mark and a destination. It is intended for real journey planning — for example, finding somewhere to stop between a departure point and an airport.

  1. Move to the starting location and press M, then 1, 2, or 3 to store it as a mark.
  2. Move to the destination and press D.
  3. Press F. The program asks which mark to use as the origin, then returns a list of food venues along the route.

Find Food is also accessible from the Tools menu (F12).

16. Tools menu

Press F12 in map mode to open the Tools menu. Tools are context-aware — some require marks, a destination, or API keys to be configured.

ToolDescription

Toll CompareCompare toll and toll-free route options for a journey.

17. Geography challenge game

The challenge game tests your knowledge of world geography by sound. The program names a country and you must locate it on the world map using movement and audio cues.

KeyAction
F10Start or stop the challenge
Ctrl+F10Start a scored challenge session
Shift+F10Repeat the current target and time remaining
RRepeat the target country name while challenge is active
Note: Jump (J) is disabled during the challenge.

Audio cues

SoundMeaning
Map position toneThe normal spatial tone — tells you where you currently are on the world map.
Sonar pulseA centred pulse that rises in pitch as you approach the target country.
HeartbeatA low rhythmic beat that speeds up as the time limit approaches.
FanfarePlays when you successfully locate the target country.
BuzzerPlays when time runs out.

Direction mode

ModeDescription
Map learningDirection feedback follows the standard left-right world map layout. Best for learning where countries appear on a flat map.
Shortest globeDirection uses the shortest great-circle route, which may cross the International Date Line. More geographically accurate.

18. Settings

Press Ctrl+, to open Settings. The dialog has three tabs: General, API Keys, and Logging.

Automatic updates

Map in a Box checks for updates automatically each time it starts. If a newer version is available, a dialog will appear asking whether you want to update. On Windows, confirming downloads and launches the installer automatically. On macOS, confirming opens the releases page in your browser to download the update manually. If there is no internet connection at startup, the check fails silently and the app continues as normal.

19. Internet and API keys

Core map features work from local bundled data and do not require an internet connection. The following features require internet access:

  • Weather and air quality
  • Timezone and sunrise/sunset
  • Wikipedia summaries
  • Country facts (live data)
  • POI lookups and detail
  • Street data downloads
  • Transit timetable data
  • Route and navigation calculation
  • Flight and hotel search
  • Satellite and Street View imagery
  • Overhead flight information

API keys and what they unlock

KeyWhere to get itWhat it enables
GoogleGoogle Cloud ConsoleGoogle Maps navigation, satellite imagery, Street View, Google-based geocoding and routing
GeminiGoogle AI Studio (free)Verbal descriptions of satellite and street view images, POI summaries, transit questions
HEREHERE DeveloperRicher live POI details, HERE walking routes, HERE departure board
OpenRouteServiceopenrouteservice.orgWalking and driving distance calculation between marks
AviationStackaviationstack.comAirport arrival and departure boards, live flight information
OpenSkyopensky-network.org (free)Overhead flight destination lookup
RapidAPIrapidapi.comFlight search (Timetable Lookup API) and hotel search (Priceline API)

What changes when a key is missing

Missing keyEffect
GoogleGoogle Maps navigation, satellite imagery, and Street View are unavailable. Street View falls back to Mapillary where possible. Other features use non-Google sources.
GeminiNo verbal descriptions for satellite or street view images, POI summaries, or transit questions. The underlying image or feature may still be available without description.
HEREPOI detail is limited to OpenStreetMap data. HERE navigation and HERE departure board are unavailable.
OpenRouteServiceDistance calculations between marks are unavailable.
AviationStackAirport arrival and departure boards are unavailable.
OpenSkyOverhead flight destination lookup is unavailable.
RapidAPIFlight search and hotel search are unavailable.

When a key is missing and a fallback exists, Map in a Box uses the fallback silently. When no fallback is available, the program explains what is needed.

20. Common workflows

Find a city

  1. Press J.
  2. Type the city name and press Enter.
  3. Choose from the results list if more than one match is found.
  4. Press Enter to jump.

Get facts about a country

  1. Move the cursor onto the country.
  2. Press F6 for area, population, and regional data.
  3. Press Shift+F1 for the capital city.
  4. Press L for languages, $ for currency.

Explore a neighbourhood at street level

  1. Jump to the city with J.
  2. Press F11 to enter street mode. Wait for data to load.
  3. Use arrow keys to move, S to search for a street by name, P for nearby POIs, A for address lookup, and G for walking directions.

Plan food stops along a route

  1. Move to the departure point. Press M then 1.
  2. Move to the destination. Press D.
  3. Press F. Choose mark 1 as the origin. Review the list of food options along the route.

Compare pickup points for a single destination

  1. Move to each pickup point and press M, storing each as mark 1, 2, and 3.
  2. Move to the destination and press D.
  3. Press Alt+1, Alt+2, and Alt+3 to compare distance and bearing from each mark.

Play the geography challenge

  1. Press F10. The program names a target country.
  2. Move with the arrow keys. Use the map-position tone to track where you are and the sonar pulse to judge how close you are to the target.
  3. Press Shift+F10 or R at any time to hear the target name again.

21. Troubleshooting

ProblemWhat to try
A key press does nothingCheck which mode you are in. Some keys are mode-specific. Press F1 for context-sensitive help.
Street mode says it is loadingWait a moment. Some areas require data to be downloaded over the internet before they can be explored.
A feature reports a missing API keyOpen Settings with Ctrl+,, go to the API Keys tab, and enter the relevant key.
The map announces "remote area"The cursor is in a sparsely populated location. Try postcode lookup, nearby features (N), or street mode if data is available for the area.
The challenge direction is confusingIn Settings, switch Challenge direction to Map learning. Also check that Spatial tones is set to World, which is easiest for global orientation.
You are lost or disorientedPress F2 to repeat your location. Press J to jump to a known place. Press Ctrl+F to open favourites.
Log file locationOpen Settings and press Open Settings Folder. The file miab.log is in that folder.
